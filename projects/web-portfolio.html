Web Portfolio

Popis projektu

Tento projekt je můj osobní web, který slouží jako portfolio mých dovedností a projektů. Využívá moderní designové trendy, jako je glassmorphism, a je plně responzivní pro mobily i desktopy. Kód je napsaný v čistém HTML a CSS s přidaným JavaScriptem pro interaktivní prvky, jako je plynulé posouvání a 3D efekty na kartách dovedností.

Technologie

  • HTML5
  • CSS3 (včetně Grid a Flexbox)
  • JavaScript (pro animace a interakce)
